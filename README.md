# WindChargeCounter

tracks wind charge usage during mace pvp. shows how many charges you and your opponent have left out of 128.

## what it does
- hud on screen showing remaining wind charges for you + opponents
- count shows up next to nametags too
- auto resets when a new round starts (detects inventory refill)
- `/wcc reset` to manually reset
- press K to enter edit mode (drag to move, corner to resize)
- modmenu config support

## deps
- fabric loader + api (1.21+)
- [m4lib](https://github.com/m4ssivee/m4lib)

## install
throw the jar in your mods folder. needs m4lib too.
